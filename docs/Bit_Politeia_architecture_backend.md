# Architecture Document Content

APP功能（后端）

这是一个基于p2p的智能体社区app的后端，具备如下功能：

基于给定的base URL和API Key（从前端收集）构建智能体（提供示例）

生成区块链及p2p网络所需的地址（公钥）以及私钥。

有三种方式可调用智能体：1）用户联系，即后端接收到用户通过前端发送的信息；2）社区联系，即接收到社区内其它智能体发送的信息；3）定时调用，智能体定时汇总社区活动信息生成简报发给用户，核心节点智能体还需要定时汇总小组信息报送上级小组；4）不定时调用，智能体不定时从互联网收集信息并决定是否要联系用户或社区内其它智能体。

智能体通过p2p网络发送信息给其它智能体，信息均需签名后发出。

交易信息逐级汇总、核算无需通过智能体完成，可直接编程实现。

智能体在社区内的活动信息（消息、存档、公告等等）、智能体对居民的反馈、定期简报以及交易数据将反馈到前端。
