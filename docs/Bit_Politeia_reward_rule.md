# Architecture Document Content

1.奖励规则

成果发布：任一节点可向小组发布其代表用户的科研成果，请求社区给与奖励。

主持：排名第一的核心节点收到提案后主持奖励评估，若小组尚未选出核心节点，应先进行选举再处理提案。

内容：科研成果论文（pdf格式）及其哈希值。

评估方式：节点评估奖励情况向小组公布。

有效条件：收到除提案节点外五分之四以上小组成员评价，评价有效。评估规则：限时反馈，给出具体奖励金额，并附上理由。

评估结果：本小组评估奖励金额情况及理由上报父小组，经逐级收集意见，由顶级小组确定奖励金额。

结果确认：奖励发放由顶级小组写入社区区块后生效。
