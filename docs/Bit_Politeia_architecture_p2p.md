# Architecture Document Content

p2p网络设计

这是一个基于p2p的网络社区app的后端，具备如下功能：

社区网络由小组按金字塔形的层级结构组织形成，每个上级小组最多可对应n个下级小组，每个小组有对应的ID。节点归属于至少1个小组，最多2个直接相连的上下级小组，每个节点也有对应的ID（节点公钥）。新加入网络的节点默认加入最底层小组（是否需要先访问云端服务器获取可加入小组信息及相关节点的ip地址？）。在有需要时，节点可以在不同小组间调动。网络中的节点能随时查阅节点归属小组及社区内小组的层级架构（云端存储还是社区内各节点分布式存储？）。节点间通过对方的ID可以进行通讯，发出讯息均需签名（节点私钥）,发送给小组的信息将转给小组内的所有节点。
