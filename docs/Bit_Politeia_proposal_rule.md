# Architecture Document Content

1.提案规则

提案：任一节点向小组发布需要投票的提案。

主持：排名第一的核心节点收到提案后指定任意核心节点主持投票，若小组尚未选出核心节点，应先进行选举再处理提案。

内容：设立新的规则或修改原有规则。

投票方式：节点投票情况向小组公布。

选举条件：收到五分之四以上小组成员选票，投票有效。投票规则：限时反馈，可投同意、不同意或弃权，并附上理由。

投票结果：得赞成票超半数提案通过，主持人宣布。

结果确认：提案通过后由核心节点确定提案的适用范围（仅适用本小组还是包括下属小组）及时间；投票结果须报请父小组（如有）审核确认，经确认后生效。
