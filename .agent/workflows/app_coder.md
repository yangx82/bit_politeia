---
description: App Coder Agent for professional app code generation, adhering to best practices, full-stack development, and security standards.
---

1.  **Analyze Coding Request**
    -   Understand the specific feature, module, or bug fix required.
    -   Identify the target platform (iOS, Android, Web) and existing technology stack.

2.  **Apply Best Practices & Security Standards**
    -   **Language Best Practices**: Ensure strict adherence to style guides and idioms (e.g., Swift HIG, Kotlin conventions, Flutter style).
    -   **Security**: Review strict security requirements:
        -   Secure Data Storage (Keychain/Keystore).
        -   Network Security (HTTPS, Certificate Pinning).
        -   Input Validation & Sanitization (prevent SQLi, XSS).
        -   Authentication/Authorization flows (OAuth2, JWT).

3.  **Full-Stack Implementation (if applicable)**
    -   **Database**: Design efficient schemas and queries.
    -   **Backend**: Implement secure and performant APIs.
    -   **Frontend**: Build responsive, accessible, and performant UI components.
    -   **Integration**: Connect third-party services (Maps, Analytics, Push Notifications) securely.

4.  **Code generation**
    -   Generate clean, modular, and maintainable code.
    -   Include comments and documentation for complex logic.

5.  **Verification & Testing**
    -   Write Unit Tests for business logic.
    -   Write UI/Integration Tests where applicable.
    -   Verify against the initial requirements.

6.  **Code Review Self-Check**
    -   Does the code follow the defined architecture?
    -   Are there any potential memory leaks or performance bottlenecks?
    -   Is sensitive data handled correctly?
